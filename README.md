# Lena 图像处理实验（课程项目）

项目背景

本项目为本人在校期间完成的图像处理课程期末作业，
基于经典 Lena 测试图像，实现并验证了多种传统图像处理算法。

项目目标并非追求工业级性能，而是通过完整实验流程，理解经典图像处理方法的原理、适用场景及其局限性，作为计算机视觉与 AI 方向的基础训练。

## 项目定位说明


本项目为 课程实验 / 教学型项目
使用经典 Lena 图像作为标准测试样例。

用于课程学习与算法验证

用于展示传统图像处理的完整处理流程

不涉及深度学习模型或训练过程

**本项目不伪装为生产系统，也不回避其课程属性。**

## 使用的数据说明


测试图像：Lena（Lenna）

用途：图像处理与计算机视觉领域的经典标准测试图像

应用场景：算法对比、效果展示、教学与实验验证

Lena 图像被广泛用于：

图像增强效果对比

噪声处理与滤波实验

图像分割与特征提取验证

## 项目结构
 ~~~
图形处理/
|--image_processing_output/ #输出的图像结果
  |-- original_rgb.jpg          # 原始RGB图像
  |-- image_info.png            # 图像信息图
  |-- enhancement.png           # 增强对比图
  |-- restoration.png           # 复原对比图
  |-- features.png              # 特征提取图
  |-- segmentation.png          # 分割对比图
  ├── processing_report.txt     # 处理报告
  └── sample_cell_image.jpg     # 示例细胞图
├── image_io.py        # Lena 图像读取与基础信息展示
├── enhance.py         # 图像增强与复原实验
├── features.py        # 特征提取实验（直方图 / GLCM / 边缘）
├── segment.py         # 图像分割实验（Otsu / 区域生长）
├── demo.py            # 实验流程演示入口

各模块按实验内容拆分，便于单独运行与理解。

## 实验内容概览

### **1️.图像读取与基础信息分析**

Lena 图像尺寸、通道数、像素范围分析

RGB 与灰度图转换

### **2️.图像增强与复原实验**

全局直方图均衡化

CLAHE 自适应直方图均衡化

椒盐噪声 / 高斯噪声模拟

中值滤波、高斯滤波去噪

***用于观察不同增强与去噪方法对图像细节的影响***

### **3️.图像特征提取实验**

灰度直方图统计特征

灰度共生矩阵（GLCM）纹理特征

Canny / Sobel 边缘检测

***用于理解传统图像特征的表达方式***

### **4️.图像分割实验**

Otsu 自动阈值分割

区域生长法分割

***对比分割方法在不同条件下的稳定性与敏感性***
